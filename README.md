## 📌 Overview

Sentiment analysis is a fundamental NLP task used to understand emotions and opinions from text data.

In this project, a **Simple Recurrent Neural Network (RNN)** is trained on the **IMDB movie review dataset** to classify user reviews into positive or negative sentiments.

The model captures sequential dependencies in text, making it effective for understanding context and tone in reviews.
